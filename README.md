# 👑 TechQueens: AI-Powered Volunteer Recommendation System

Developed as part of the **GSB AI and Informatics Winter Camp 2025**, this project is an intelligent platform that analyzes the academic backgrounds and interests of students staying in GSB dormitories to match them with the most suitable volunteer activities using artificial intelligence.

##  Camp and Hackathon Story
This project was developed by the **TechQueens** team for the final hackathon of the **AI and Informatics Winter Camp** organized by the Ministry of Youth and Sports (GSB). Our goal is to transform the competencies we gained during the camp into a technology that adds value back to the GSB ecosystem.

##  Purpose of the Project
We connect the potential of thousands of university students residing in GSB dormitories with the right social responsibility projects:
- **Personalized Matching:** Department-oriented assignments (e.g., a coding workshop for a software engineering student).
- **Social Development:** Strengthening students' portfolios while increasing their social responsibility awareness.
- **Continuity:** Encouraging long-term volunteering through gamification elements.

##  How It Works (Technical Infrastructure)
The system utilizes modern data processing techniques presented to the jury:
1. **Data Processing:** Student profiles are analyzed using NLP (Natural Language Processing).
2. **AI Engine:** The most compatible projects are scored using the "Content-Based Filtering" algorithm.
3. **Smart Notification:** The top 3 activities with the highest matching scores are recommended to the user.

##  Highlighted Features
-  **Volunteering Scorecard:** A digital competency certificate generated after each completed activity.
-  **Gamification:** Leadership tables and competition between different dormitories.
-  **Department Integration:** Opportunity to put theoretical knowledge into practice in the field.

##  Technologies
- **Backend:** Python (FastAPI / Flask)
- **AI/ML:** Scikit-learn, Pandas, NLTK
- **Interface:** Flutter / React Native
